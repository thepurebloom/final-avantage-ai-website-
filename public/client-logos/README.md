# Client Logos Directory

## Instructions

Add your client logo images to this folder with the following naming convention:

- `client-1.png`
- `client-2.png`
- `client-3.png`
- `client-4.png`
- `client-5.png`
- `client-6.png`
- `client-7.png`
- `client-8.png`

## Image Requirements

- **Format**: PNG with transparent background (recommended) or JPG
- **Size**: Recommended width: 200-400px, Height: 80-120px
- **Aspect Ratio**: Horizontal logos work best (landscape orientation)
- **Background**: Transparent backgrounds are preferred
- **Color**: White or light-colored logos work best on the dark background

## Customization

To customize the carousel, edit `/components/ui/logos-carousel.tsx`:

1. Update the `DEFAULT_LOGOS` array to change logo paths
2. Modify the `heading` prop when using the component
3. Adjust `className` values for logo sizing
4. Change carousel speed in the `AutoScroll` plugin configuration

## Usage

The logos will automatically appear in the animated carousel on your homepage below the "Companies We Collaborate With" section.
